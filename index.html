css image-set()

It is a promising method to use for retina sprite images.

.imageSetIcons {
    background-image: url('icons-1x.png');
    background-image: -o-image-set(url('icons-1x.png') 1x, url('icons-2x.png') 2x);
    background-image: -moz-image-set(url('icons-1x.png') 1x, url('icons-2x.png') 2x);
    background-image: -webkit-image-set(url('icons-1x.png') 1x, url('icons-2x.png') 2x);
}

Wordpress post thumbnails

From wordPress 4.4 you have wp_get_attachment_image_srcset() and wp_get_attachment_image_sizes() to get the dynamic post images.

<?php
	$attachment_id = get_post_thumbnail_id( $post_id );
	$img_src = wp_get_attachment_image_url( $attachment_id, 'large' );
	$img_srcset = wp_get_attachment_image_srcset( $attachment_id, 'medium' );
	$img_sizes = wp_get_attachment_image_sizes( $attachment_id, 'large' );
?>

Standard instructions

  • Maximum 1600w for large screens and 750w for small screens
  • Retina icons for mobile screens
  • File size must be below to 100kb and compress all the images
  • Use sprite images/fonts for icons

So...

  • <picture> is used for art direction and different image types
  • srcset to get the best source based on cache and resolution
  • sizes to get the image, based on its own resolution than viewport
  • image-set() is for retina sprite images
  • webp for the larger file size image (more than 300kb), Online convertor.

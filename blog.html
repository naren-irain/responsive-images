Responsive images

- Images has the important role in the content representation in most of the websites - And it is the largest part in a web page interms of the page size and the load time apart from video and audio. - In the modern world there are different kind of displays and screen resolutions. - Showing the same image for all these screens are not ideally good. - This responsive images technique helps us to display the best image based on the device and screen capabilities.

- We have lot of responsive images techniques, through CSS, picture tag, srcset. - Now we are going to discover picture tag, srcset, images-set() - We need to choose the best among the other options.

- Before that, look at the history of the responsive images integration - Setting max-width: 100%; for tag. so, it will scale-up and scale-down to the container size - Changing background-image in media-queries - Fit the images with background-size - Handling icons in the fonts like fontawesome, icomoon or bootstrap glyphicons - Now, we have more controls with picture tag, srcset and src-set(). Lets start about these

picture>

- The picture tag is very similar in the concept to the audio and video elements. But in picture we need both sources and img tag. - First there will be set of sources with some conditions to display. - srcset attrubute to give the source file path - media attribute for media query conditions - type attribute for image type declaration - A standard img tag is the last required element.

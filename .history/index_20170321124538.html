CSS Options

Media queries

@media only screen and (max-width: 750px) {
    .cssBanner { background-image: url('../img/car-small.jpg'); }
}
@media only screen and (min-width: 751px) {
    .cssBanner { background-image: url('../img/car-large.jpg'); }
}

css image-set()

.imageSetIcons {
    background-image: url('icons-1x.png');
    background-image: -webkit-image-set(url('icons-1x.png') 1x, url('icons-2x.png') 2x);
}

Browser support - image-set()

css image() function

.webpBanner {
    background-image: url('../img/clubmed.jpg');
    background-image: url('../img/clubmed.webp');
    background-image: image('../img/clubmed.webp', '../img/clubmed.jpg');
}

Wordpress post thumbnails

From wordPress 4.4 you have wp_get_attachment_image_srcset() and wp_get_attachment_image_sizes() to get the dynamic post images.

<?php
$img_src = wp_get_attachment_image_url( $attachment_id, 'medium' );
$img_srcset = wp_get_attachment_image_srcset( $attachment_id, 'medium' );
?>
